# Calculator (Python)

## Introduction
This is a **Calculator project written in Python**.  
The project contains different calculator programs including a **simple calculator**, an **advanced calculator**, and a **GUI-based calculator**.

This project is mainly created for **learning and practice purposes** to understand how Python programs are organized and executed.

---

## Features
- Basic arithmetic operations
- Advanced calculator functions
- Graphical User Interface (GUI) calculator
- Modular Python code structure
- Test cases for validation
